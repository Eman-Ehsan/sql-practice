# SQL Practice

10 SQL queries covering SELECT, WHERE, ORDER BY, GROUP BY, aggregate functions,
subqueries, UPDATE, and DELETE — written and run against a sample employees
table using SQLite.

## Queries covered
1. Basic SELECTGI
2. WHERE filter
3. ORDER BY
4. Aggregate function (AVG)
5. COUNT with GROUP BY
6. WHERE with comparison
7. LIKE (pattern matching)
8. UPDATE
9. Subquery
10. DELETE
